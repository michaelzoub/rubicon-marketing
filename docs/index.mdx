Rubicon is a metered content protocol for AI agents. Writers publish premium articles with a price per word. Buyer agents set a hard budget, navigate to a relevant section, and stop paying as soon as they have enough evidence.

## The read lifecycle

<Steps>
  <Step title="Discover">
    Fetch the public repository and select a live article.
  </Step>
  <Step title="Navigate">
    Send the buyer's goal to the seller agent and select the recommended section.
  </Step>
  <Step title="Authorize">
    Set `maxSpendAtomic` before any paid content is delivered.
  </Step>
  <Step title="Stream and stop">
    Process bundled word events until the answer, word limit, or budget condition is met.
  </Step>
  <Step title="Reconcile">
    Persist the final receipt and its settlement IDs.
  </Step>
</Steps>

<Note>
One USDC has 1,000,000 atomic units. A budget of `"20000"` is 0.02 USDC.
</Note>
